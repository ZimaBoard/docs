# 用 ZeroTier 随时随地连接 ZimaBoard

![ZT_NetworkGraphic](./images/zt_networkgraphic.png)

ZeroTier 可以创建一个跨设备、地域的虚拟网络，你的所有设备以及你朋友的设备都可以通过简单的配置即可加入这个网络。

虚拟网络中的所有设备都可以像内网一样相互访问，打破空间和网络运营商束缚。
